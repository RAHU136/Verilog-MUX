# 2:1 Multiplexer Design using Verilog HDL

## Project Overview
This project implements a 2:1 Multiplexer (MUX) using Verilog HDL and verifies its functionality through simulation.

## Files Included

### mux.v
Contains the Verilog design of the 2:1 Multiplexer.

### tb_mux.v
Contains the testbench used for verification and simulation.
